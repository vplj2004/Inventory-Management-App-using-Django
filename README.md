## Inventory Management App

This is a Django-based web application for managing product inventories. It allows users to add products with various attributes and an image, and displays a list of all added products along with their images. The app utilizes `crispy-bootstrap5` and `crispy-forms` for a responsive and user-friendly interface. The `Pillow` library is required for image handling.

### Features:
- Add products with customizable attributes and upload images.
- View a list of all products with their corresponding images.
- Responsive design using Bootstrap 5.

### Requirements:
- Django
- crispy-bootstrap5
- crispy-forms
- Pillow
